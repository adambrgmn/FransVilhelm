# useOnline

Determine if the client is online or offline and update the value if the state
changes.

## Usage

```jsx
import React from 'react';
import { useOnline } from '@fransvilhelm/hooks';

const ConnectionIndicator = () => {
  const online = useOnline();
  return (
    <div>
      <CircleIcon color={online ? 'green' : 'red'} />
      {online ? 'connected' : 'disconnected'}
    </div>
  );
};
```

## Parameters

`useOnline` accepts no parameters

## Returns

`useOnline` returns a boolean indicating if the client is online or not.

| Returns | Type    | Description                       |
| :------ | :------ | :-------------------------------- |
| online  | boolean | If the client is connected or not |
